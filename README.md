# 🤖 Telegram-бот для проверки ревью на dvmn.org

Бот отслеживает изменения статуса проверок ваших работ на платформе [dvmn.org](https://dvmn.org/) и отправляет уведомления в Telegram при получении нового ревью.

## 📦 Возможности

- 🔍 Отслеживание новых проверок работ через API dvmn.org
- 📨 Уведомления в Telegram о результатах проверки (положительные/отрицательные)
- ⏳ Долгий опрос (long polling) для получения обновлений в реальном времени
- 📝Логирование событий в файл `sample.log`

## 🧰 Требования

- 🐍 Python 3.8 до 3.10 более новые версии не поддерживаются

### Инструкция регистрации для [Бота](https://way23.ru/%D1%80%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F-%D0%B1%D0%BE%D1%82%D0%B0-%D0%B2-telegram.html)

И получения его 💰`TG_BOT_TOKEN` например 958423683:AAEAtJ5Lde5YYfkjergber  
Для получения `TG_CHAT_ID` воспользуйтесь @userinfobot (ботом в телеграме)

## 🧱 Установка

** Установите зависимости: **

```bash
pip install -r requirements.txt
```

**Часть настроек проекта берётся из переменных окружения.**  
**Чтобы их определить, создайте файл .env (точка в начале обязательна!) рядом с main.py**  
**и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.**

```
TG_BOT_TOKEN=ваш_токен_бота_telegram
DEVMAN_TOKEN=ваш_токен_api_dvmn
TG_USERNAME=ваш_никнейм_telegram
TG_CHAT_ID=id_чата_для_уведомлений
```

## 🚀 Запуск

```bash
python main.py --id <chat_id>
```

## ⚙️ Конфигурация

Файл `.env` должен содержать следующие переменные:

| Переменная     | Описание                                                            |
| -------------- | ------------------------------------------------------------------- |
| `TG_BOT_TOKEN` | Токен вашего Telegram-бота                                          |
| `DEVMAN_TOKEN` | Токен API dvmn.org                                                  |
| `TG_USERNAME`  | Ваш никнейм в Telegram (без @)                                      |
| `TG_CHAT_ID`   | ID чата для уведомлений (опционально, если не передан через `--id`) |
